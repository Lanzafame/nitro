## Install

With [fisherman]

```fish
fisher nitro
```

## Features

* Git
    * Clean
    * Dirty / Touched
    * Staged
    * Staged + Dirty
    * Stashed
    * Unpushed commits (ahead)
    * Unpulled commits (behind)
    * Unpulled and unpushed commits (diverged)
    * Detached HEAD
    * Branch name
* $status
* $CMD_DURATION
* $VIRTUAL_ENV
* $status in context
* Background jobs
* Superpowers (sudo)
* Host information
